# Анализ временного ряда Air Passengers
## Цель работы
Провести полный цикл анализа временного ряда (EDA, моделирование, оценка), сравнить статистические методы (SeasonalNaive, SARIMA, AutoETS, Theta) и методы машинного/глубокого обучения (LightGBM, XGBoost, Random Forest, DeepAR, N-BEATS, TFT), построить пайплайн прогнозирования и дать обоснованные рекомендации.

## 1. Исследовательский анализ данных (EDA)
### 1.1 Общая информация
- Период: январь 1949 – декабрь 1960 (144 наблюдения)
- Частота: ежемесячная
- Пропуски отсутствуют

### 1.2 Визуальный анализ
*График исходного ряда:* явный восходящий тренд и выраженная годовая сезонность с увеличивающейся амплитудой (мультипликативный характер).

*Декомпозиция (мультипликативная):* тренд, сезонность и остатки выделены явно. Остатки близки к нормальному распределению (тест Шапиро–Уилка p-value = 0.0229, тест Бройша–Пагана подтверждает гомоскедастичность).

*Коррелограммы:* ACF затухает медленно, PACF имеет значимый пик на лаге 1 – признаки нестационарности.

### 1.3 Проверка стационарности
| Ряд | ADF p-value | KPSS p-value | Вывод |
|-----|-------------|--------------|-------|
| Исходный | 0.9919 | 0.01 | Нестационарен |
| Логарифм | 0.4224 | 0.01 | Нестационарен |
| Первая разность логарифма | 0.0711 | 0.10 | Стационарен по KPSS |
| Сезонная разность логарифма | 0.0724 | 0.10 | Стационарен по KPSS |
| Комбинированная разность | 0.0002 | 0.10 | **Стационарен** |

**Вывод:** для приведения ряда к стационарности необходимо логарифмирование + сезонное дифференцирование (12) + обычное дифференцирование.

## 3. Пайплайн и дополнительное тестирование
### 3.1 Кросс-валидация (TimeSeriesSplit, 5 блоков)
Результаты средней MAE:
- AutoARIMA: 31.9
- LGBM: 46.8

Статистическая модель стабильнее на истории.

### 3.2 Тест Дибольда–Мариано (сравнение AutoARIMA и LGBM)
Статистика DM = 2.82, p-value = 0.0048 → различия статистически значимы, LGBM лучше на тесте.

### 3.3 Производительность
- AutoARIMA: обучение + прогноз ~ 0.01 сек
- LGBM: обучение + прогноз ~ 0.06 сек

## 4. Общие выводы и рекомендации
- Ряд обладает сильной мультипликативной сезонностью и трендом, требует логарифмирования и дифференцирования для стационарности.
- Среди всех моделей **LightGBM** показал наименьшую ошибку (MAE = 11.81), но статистическая модель **AutoARIMA** значительно быстрее и даёт близкое качество (MAE = 18.52). Различие статистически значимо в пользу LightGBM.
- Глубокое обучение (NBEATS, TFT) уступает как по точности, так и по скорости из-за малого объёма данных.
- **Рекомендация для внедрения:** для данного объёма данных оптимален **AutoARIMA** – высокая скорость, приемлемая точность, интерпретируемость. Если требуется максимальная точность и есть вычислительные ресурсы, можно использовать **LightGBM** с тщательной настройкой гиперпараметров.
